# Perfume Blending Calculator

Free perfume blending calculator for DIY perfumers. Built this to help formulators get top, middle and base note ratios right without doing the math manually.

**Live calculator:** [perfume-calculator.netlify.app](https://perfume-calculator.netlify.app)

## What it does
- Top, middle, base note ratio calculator
- Fragrance load calculator by product type
- Batch size breakdown (exact grams per oil)
- Safe dilution percentage reference
